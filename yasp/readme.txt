YASP (Yet Another Stats Plugin) displays some useful numbers about your blog/website on your admin panel dashboard.

== Description ==

YASP (Yet Another Stats Plugin) is an admin panel widget.  You can add it to your dashboard to see useful stats about your website/blog every time you log in.

Data available includes total number of posts, total number of comments, total number of categories, etc.

YASP is lightweight, and free from spam.

== Installation ==

Installation is easy.  

## Recommended

Searching "YASP" in your plugin menu from within WordPress, then hit Install.

## Manual

1.  Upload the `yasp` folder to your plugins directory `/wp-content/plugins/`.
2.  Activate the plugin

## Usage

YASP will then be available from your Dashboard.  If you cannot see if, check the 'Screen Options' tab at the top.

== Frequently Asked Questions ==

= I've installed YASP but I can't see it anywhere.  What's going on? =

Click the 'Screen Options' button at the top of the Dashboard and you should see a series of checkboxes.  Make sure YASP is checked.

= It's still not showing.  What gives? =

Make sure you have actually activated the plugin.  To do this go to your plugins section, and select 'Activate'
